* Description

This repo contains ZSH scripts for RF capture (video8 / 8mm) at 40MHz/40MSPS via the vhs-decode project for the follwing operations:

1. CX auto level calibration
   #+begin_src
. cx-calibrate.zsh
   #+end_src
2. FLAC RF capture
   #+begin_src
capture-rf-flac.zsh /path/to/capture-40msps-8bit-cx-card.flac
   #+end_src
3. vhs-decode video + hifi + vhs-decode-auto-audio-align + tbc-video-export FFV1 video coding, MATROSKA containter
   #+begin_src
decode-rf.zsh /path/to/capture-40msps-8bit-cx-card.flac /path/to/capture-40msps-8bit-cx-card
   #+end_src

*Note:* This repo is for my personal use / archiving. Therefore adaption on paths is required if your are attempting to reuse.
